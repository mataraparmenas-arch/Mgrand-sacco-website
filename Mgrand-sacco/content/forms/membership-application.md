# Membership Application Form

This form is required for all new members joining Mgrand Sacco. Please complete all sections and submit to any branch office.

## Required Documents

- Copy of National ID or Passport
- Passport size photo
- Proof of residence (utility bill or tenancy agreement)
- KRA PIN certificate
